# Test Audit — 2026-03-20

**Total collected:** 609 tests across 21 test files

---

## Well-covered areas

**Parsing utilities** (`tests/loaders/test_parsing.py`) — 46 tests across `TestParseFloat` (24) and `TestParseDollar` (22). Thoroughly parametrized over edge cases; these utilities underpin every loader so the coverage is appropriate.

**Validators** (`tests/test_validators.py`) — ~44 tests covering `TestValidateAccountId` (15), `TestValidateDate` (14), `TestValidateModel` (10), `TestValidateDataDir` (5). Good input-boundary coverage.

**Calculator layer** — strongest area overall. `test_budget.py`, `test_net_worth.py`, `test_debt_payoff.py`, `test_savings_rate.py`, `test_spending_trends.py` each cover unit logic and sample-data integration. Regressions in calculation math would be caught immediately.

**Usage logger / redaction** (`tests/test_usage_logger.py`) — 43 tests across `TestRedact`, `TestRedactParam`, `TestSafeArgs`, `TestLogToolCall`, `TestIsSample`. Unusually thorough for infrastructure; appropriate given this touches real financial data.

**Loaders** — `test_capital_one.py`, `test_fidelity.py`, `test_vanguard.py`, `test_qfx.py` all have dedicated files covering accounts, holdings, transactions, errors, and structure.

## Recommended next tests (priority order)

1. **`src/server.py`** — Add at least smoke-level integration tests that invoke each MCP tool through the server with sample data and assert the response shape. This is the highest-value gap: it's the only layer that connects loaders, calculators, and the MCP interface, and it has zero coverage.

2. **`src/importers/base.py` — `clean_num` and `_find_header_row`** — Now that these are module-level functions, add direct unit tests. Small effort, closes a gap introduced by the refactor.

3. **`src/importers/fidelity.py` and `src/importers/vanguard.py`** — Add dedicated `TestFidelityImporter` and `TestVanguardImporter` test classes mirroring the existing Chase and Capital One importer tests. Cover header detection, mode switching, and numeric parsing.
